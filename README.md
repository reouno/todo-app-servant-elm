# todo-app-servant-elm

# Run

```
# build backend
$ stack build

# build frontend
$ cd frontend
$ yarn install # only for the first time
$ yarn webpack

# run server
$ cd ..
$ stack exec todo-app-servant-elm-exe
```

access to [http://localhost:3000](http://localhost:3000)
